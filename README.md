# Retail ETL Pipeline Project

## Overview
This project demonstrates an end-to-end ETL pipeline using Python, PySpark, and SQL.

## Tech Stack
- Python (Pandas)
- PySpark (Databricks)
- SQL

## Project Flow
Raw Data → Pandas Cleaning → Spark Processing → Fact Table → SQL Analytics

## Folder Structure
- data/raw: Original datasets
- data/cleaned: Cleaned datasets and final fact table
- scripts: ETL scripts (Pandas + Spark)
- sql: Database schema and analytics queries
- outputs: Final reports (optional)

## How to Run
1. Run pandas_cleaning.py
2. Run spark_processing.py in Databricks
3. Execute SQL queries in Databricks SQL

## Output
- Cleaned datasets
- Fact sales table
- Business insights from SQL queries
